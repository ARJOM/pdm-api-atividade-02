 ## Executando

 1. Instale as dependências
    ```
    yarn
    #ou
    npm install
    ```
    1.2 (Opcional) Crie um arquivo `.env`na raiz do projeto para escolher a porta em que a aplicação irá escutar, e preencha nesse formato
    ```
    PORT=8085
    ```

2. Execute em modo de desenvolvimento
    ```
    yarn start:dev
    #ou
    npm run start:dev
    ```
